JHelpBuilder - Add Index Entries

Adding entries to Index is similar to adding TOC entries.

  • Go to the Index tab.
  • Select the entry below which you want to add the new entry.
  • Click the button in the toolbar.
  • A "new indexitem" entry will be added.
  • The "new indexitem" entry should be already selected. If not, select the entry.
  • The Properties panel will display the properties of this entry
  • Click the value column of "text" property and change the value to "Eggplant"
  • Click the value column of "target" property. Change the value to "eggplant_garlicky_brown_sauce_html". You can also use the "..." button to display a dialog which will assist you in selecting a target.
  • Optionally, you can provide values for other properties also.
Repeat the steps above to create an Index as shown below.
