JHelpBuilder - Generate jar file

Generating a jar file is the final step. If you need to make any changes to your helpset, you should do them before this step.

  • Click on Tools -> Jar HelpSet.
  • Select a Jar file and click Ok.
  • The jar file with your helpset will be generated. You can now use this jarred helpset in any java application.
