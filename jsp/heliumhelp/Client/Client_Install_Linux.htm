Helium V Client Installation unter Linux

Unter dem Oberbegriff Linux stehen verschiedene Distributionen zur Verfügung.

Je nach der Distribution sind unter Umständen leicht abgewandelte Vorgehensweisen bei der Installation zu beachten.

Grundsätzlich ist für die Verwendung des Helium V Clients ein grafischer Desktop Grundvoraussetzung.

 

Ubunutu

Verwenden Sie hier die Version 10.10

 

Der Aufruf der Startdatei lpclientpc.sh muss um den Verzeichniswechsel ergänzt werden. 

Zusätzlich sollte der Helium V Client nur im sogenannten Kunststoff Look betrieben werden.

Siehe Beispiel unten.

 

 

#\!/bin/sh
cd /home/ubuntu/helium_v

java -Xms128m -Xmx256m -XX:PermSize=64m -XX:MaxPermSize=256m -Djava.naming.factory.initial=org.jnp.interfaces.NamingContextFactory -Djava.naming.provider.url=jnp://localhost:2099 -Dloc=de_AT -jar lpclientpc.jar --laf-Kunststoff $*

Stellen Sie zugleich sicher, dass Sie ausreichend Rechte haben um die Installation des Helium V Clients überhaupt durchführen zu dürfen.

 

Anpassen des Erscheinungsbildes des Helium V Start Scripts

Klicken Sie auf das Symbol des Helium V Startscripts, rechte Maustaste, Eigenschaften.

Nun sehen Sie

Um das Erscheinungsbild auf das Helium V Symbol zu ändern klicken Sie direkt das Symbol an und wählen nun aus dem Helium V Ordner, der üblicherweise unter home/benutzername/helium_v zu finden ist das

Symbol lpclientpc.png, welches das Helium V Icon beinhaltet.
