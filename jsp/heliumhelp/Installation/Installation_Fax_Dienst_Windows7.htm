Installation des Faxdienstes auf Windows7

Um die Faxdienst unter Windows 7 zu installieren gehen Sie bitte in die Systemsteuerung und wählen Sie Programme.
Nun wählen Sie Windows-Funktionen aktivieren oder deaktivieren



Erweitern Sie Druck- und Dokumentdienste und stellen Sie sicher, dass Windows-Fax und -Scan angehakt ist.
Gegebenenfalls muss der Faxdienst nachinstalliert werden.

Testen des Faxversandes

Öffnen Sie einen Texteditor, z.B. Wordpad oder auch Notepad (Editor). Beide finden Sie unter Start, Programme, Zubehör.

Geben Sie einen Text ein und drucken Sie diesen an den Faxdrucker.
