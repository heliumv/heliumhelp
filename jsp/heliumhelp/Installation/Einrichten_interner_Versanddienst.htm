Einrichten interner Versanddienst

Mit dem internen Versanddienst steht Ihnen aber Version 5956 eine direkte SMTP und IMAP Anbindung für den Versand Ihrer EMails und gegebenfalls auch Faxe direkt integriert im Helium V Applikation Server zur Verfügung. D.h. der Umweg über Windows wird hiermit nicht mehr benötigt.

Die Parametrierung des internen Versanddienstes erfolgt im Modul System. Geben Sie hier unter Kategorie Versand ein, so werden die erforderlichen Parameter für den internen Versanddienst gelistet.

Sollten Sie in Ihrer Installation diese Parameter nicht vorfinden, so ist in Ihrer Helium V Installation dieses Modul nicht freigeschaltet. Bitte wenden Sie sich an Ihren Helium V Betreuer.

Einrichten des SMTP Servers:

Durch die Parametrierung des SMTP Servers, wird der interne Versanddienst aktiviert.

Parameter

Beispiel

Beschreibung

SMPTSERVER

mail.logistikpur.local

Servername des SMPT Server

SMTPSERVER_BENUTZER

SMTPSERVER_FAXDOMAIN

fax.local

SMTPSERVER_KENNWORT

MAILADRESSE_ADMIN

Einrichten der IMAP Anbindung:

Durch die IMAP Anbindung, wird nach erfolgtem SMTP Versand eine Kopie des EMails im IMAP Postfach des Benutzers abgelegt. Die Parametrierung ist zweigeteilt:

a.) die generelle Einbindung, diese finden Sie ebenfalls unter System, Parameter, Kategorie Versand

b.) die Definition der Postfächer für jeden Benutzer, diese finden Sie im Modul Personal, im oberen Modulreiter Daten.

 

Parametrierung IMAP Anbindung

Parameter

Beispiel

Beschreibung

IMAPSERVER

mail.logistikpur.local

Servername des IMAP Server

IMAPSERVER_ADMIN
