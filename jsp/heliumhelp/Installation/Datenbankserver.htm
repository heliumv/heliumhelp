Datenbankserver

Von HELIUM V werden derzeit folgende Datenbankserver unterstützt:

  • PostgresQL Version 8
    Diese Lizenzfreie Version wird mit Ihrer Linux Installation mitgeliefert. In Ihrer MAC Installation wird diese von Apple zur Verfügung gestellt.

  • Mircosoft MSDE Freie Version für Mircrosoft Betriebssysteme.
    Beachten Sie bitte die Größenbegrenzung dieser für den Einsatz mit Microsoft Betriebssystemen kostenlosen Datenbank auf 2GByte.

  • Mircosoft MSSQL Server 2000

  • Microsoft SQL Server 2005 (Express) Diese freie Version ist auf 4GB begrenzt und muss vom Anwenderunternehmen installiert werden.

  • Microsoft SQL Server 2008

  • Microsoft SQL Server 2008 (Express) Diese freie Version ist auf 4GB begrenzt und muss vom Anwenderunternehmen installiert werden.

Je nach Größe der zu erwartenden Daten, raten wir bereits ab Beginn 64Bit Versionen von Betriebssystem und Datenbankserver zu verwenden.

Hinweis:

Von Seiten HELIUM V werden für den Zugang zum Server nur zwei Benutzer  verwendet, egal wieviele gleichzeitige Benutzer in Ihrem HELIUM V arbeiten. Beim Einsatz der Datenbankserver ist die zukünftige Größenentwicklung der Datenbank und die eventuellen zusätzlichen auswertenden Zugriffe, welche ev. außerhalb von HELIUM V bzw. in manchen Spezialabfragen aus den Reports von Ihren Mitarbeitern bereitgestellt werden zu beachten. Denken Sie bei der Größenentwicklung auch an die Dokumentenablage.

Diese Daten werden in einer eigenen Datenbank neben den ERP Daten Ihres HELIUM V Systems abgelegt.

Sicherung der Datenbank:

Bitte stellen Sie unter allen Umständen sicher, dass Sie zumindest ein tagesaktuelles Backup Ihre Daten und Ihrer Formularvorlagen, Reports haben. Mit diesen Daten können wir Ihnen im Falle des Falles jederzeit helfen. Ohne diesen Daten sind wir machtlos.

Wir benötigen für eine Wiederherstellung die Datensicherung der Datenbank LP, der LP_DOKUMENTE und der Reports.

 

Denken Sie hier auch an die Ergebnisse ausführlicher Studien, die besagen, dass EDV gestützte Unternehmen, mit einem Datenverlust / nicht verfügbaren Daten von nur drei Tagen zum Großteil nach einem Jahr in die Insolvenz geschlittert sind.

Speichereinstellungen

Für den HELIUM V Server werden, abhängig vom Betriebssystem und der verwendeten Java VM (JDK) folgende Einstellungen verwendet. Bitte beachten Sie, dass für die Clients eigene Einstellungen erforderlich sind.

Betriebssystem Speicher Xms Xmx
32Bit, JDK 1.6.0_5 bis 1.6.0_xx 2GB 512m 768m
32Bit, JDK 1.6.0_5 bis 1.6.0_xx 4GB 768m 1280m
64Bit, JDK 1.6.0_5 bis 1.6.0_xx (64Bit) 4GB 768m 1280m
64Bit, JDK 1.6.0_5 bis 1.6.0_xx (64Bit) 16GB 2048m 4096m
64Bit, JDK 1.6.0_5 bis 1.6.0_xx (64Bit) 32GB 4096m 8192m
