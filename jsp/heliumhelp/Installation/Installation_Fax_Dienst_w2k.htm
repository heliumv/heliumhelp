Installation des Faxdienstes auf einem Windows2000 Server

 

Ist bei der Basis Installation des Windows2000 Servers kein Modem angeschlossen, so wird auch der Faxdienst nicht installiert. D.h. für die zur Verfügungstellung des FaxPrinters muss zuerst ein Modem installiert werden. Egal ob dieses tatsächlich vorhanden ist oder nicht.
Gehen Sie dazu wie folgt vor:
Rechtsklick auf Arbeitsplatz, Eigenschaften, Hardware, Neue Hardware suchen.
Am Schluss erscheint eine Auswahlliste, wählen Sie hier, Neues Gerät hinzufügen.
Hardwarekomponenten aus Liste auswählen
Modems, haken Sie Modem auswählen an.
Wählen Sie ein Modem, z.B. Elsa Microlink 56k
Wählen Sie den Anschluss, z.B. Com1. Achten Sie dabei darauf dass dieser Anschluss nicht von einem anderen Gerät verwendet wird.
Nun gehen Sie in die Systemsteuerung, wählen Sie Telefon und Modemoptionen und konfigurieren Sie den Standort.
Nach erfolgter Konfiguration steht, eventuell nach Aktualisierung der Systemsteuerung (F5), der Faxdrucker zur Verfügung.

Nun benötigen Sie noch die Datei FXSAPI.dll. Diese finden Sie auf Ihrer Helium V CD unter "helium-v_versand\Windows2000_FaxDateien". Kopieren Sie diese Datei nach ?:\winnt\system32

Hinweis:

Windows 2000 erstellt den Fax-Drucker nur unter zwei Bedingungen:

a.) Wenn ein Modem (nachträglich) installiert wird (siehe oben)

b.) Wenn während der Installation von Windows 2000 bereits ein Modem angeschlossen und eingeschaltet ist.

 

Testen des Faxversandes unter Windows 2000

Öffnen Sie einen Texteditor, z.B. Wordpad oder auch Notepad (Editor). Beide finden Sie unter Start, Programme, Zubehör.

Geben Sie einen Text ein und drucken Sie diesen an den Faxdrucker.
