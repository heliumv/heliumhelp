Bedienung der HELIUM V Hilfe

In die HELIUM V Hilfe gelangen Sie nach der Anmeldung über den Menüpunkt Hilfe, HELIUM V Hilfe im HELIUM V Client.

 

Nach dem Einstieg in die Hilfe maximieren Sie bitte das Fenster.

 

Sie finden nun links den Index und die Textsuche .

Durch Klick auf den Index sehen Sie alle Kapitel. Gegebenfalls klappen Sie die Kapitel durch Klick auf auf.

In den Kapiteln können Sie mit Strg+F nach Texten suchen.

 

Sind in den Kapiteln Hinweise auf andere Kapitel enthalten, so sind diese blau hinterlegt. Klicken Sie auf diesen Link und Sie gelangen auf die entsprechende Stelle. Um wieder zurückzugelangen drücken Sie bitte auf den Zurück-Pfeil .

 

Um die Hilfe im Webbrowser aufzurufen, geben Sie bitte HELIUMVSERVER:8080/heliumhelp in der Adresszeile ein. Ersetzen Sie dabei HELIUMVSERVER bitte durch den Namen Ihres HELIUM V-Servers.

Anwenderspezifische Hilfe

 

Unter dem Menüpunkt Hilfe, Anwenderspezifische Hilfe steht Ihnen ein Link auf Ihre unternehmensspezifische Anwendung von HELIUM V zur Verfügung.

Diese Hilfe ist als HTML Datei aufgebaut. Benutzer die diese Hilfe anzeigen wollen, müssen auf diesen UNC Pfad zumindest Leserechte besitzen.

 

Wird beim Aufruf der anwenderspezifischen Hilfe

angezeigt, so wurde der Pfad nicht definiert.

 

Der Pfad auf Ihre Hilfedatei kann im System unterer Modulreiter Parameter, URL_ANWENDERSPEZIFISCHE_HILFE eingestellt werden.

Bitte achten Sie darauf hier allgemeingültige Pfade einzustellen.

 

Wichtig:

Hier muss eine URL eingegeben werden.

so wurde vermutlich nur ein Filename angegeben. Für den vermutlich üblichen Verweis auf eine Datei muss: file://Server/Freigabe/... angegeben werden. Bitte beachten Sie, dass nur die Vorwärts-Schrägstriche verwendet werden dürfen.

 

Kann die Datei nicht gefunden werden, z.B. wegen Leserechten, so erscheint

.

Bitte entsprechend richtigstellen.

 

Direkthilfe - Hilfe zu einzelnen Feldern erfassen und anzeigen

Die direkte Hilfe ist vergleichbar mit den Tool Tipps und dient der direkteren Information der Anwender von HELIUM V. Wenn Sie die Maus über ein Feld bewegen, das einen Dirkethilfe Eintrag hinterlegt hat, so wird dieser angezeigt. Das ist besonders hilfreich um schnell Informationen zu nicht so häufig verwendeten Feldern zu erhalten. 

Um die Anzeige der Direkthilfe an- oder abzuschalten drücken Sie die Taste F3.

 

Die Texte der Direkthilfe werden einerseits von HELIUM V für Sie gepflegt - andererseits können Sie auch selbst Inhalte, die für Ihr Unternehmen relevant sind eintragen.

 

Um die Hilfetexte zu bearbeiten benötigen Sie das Benutzerrecht LP_DARF_DIREKTHILFETEXTE_BEARBEITEN. Damit die roten Dreiecke nicht angezeigt werden, löschen Sie das Recht aus der Systemrolle.

Klicken Sie auf das rote Dreieck bei dem Feld, dass Sie beschreiben möchten.

Nun öffnet sich die Eingabe in ähnlicher Form und das Dreieck ändert die Farbe auf schwarz.

Nach Klick auf das Icon Bearbeiten können Sie in diesem Feld neben Text auch HTML-Code (zum Beispiel zur Formatierung) erfassen.

Wechselt man während des Editierens eines Textes in ein anderes Fenster, bleibt die Direkthilfe im Hintergrund geöffnet, der Text bleibt erhalten. Während des Editierens lässt sich kein anderer Direkthilfe-ToolTip öffnen.

Ein erneuter Klick auf Bearbeiten speichert den Eintrag ab. Um die Erfassung zu verlassen klicken Sie auf beenden .

Ein Eintrag zu einem Feld ist daran erkennbar, dass das Dreieck grün eingefärbt ist.
