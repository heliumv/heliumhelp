Bedienung der HELIUM V Hilfe

In die HELIUM V Hilfe gelangen Sie nach der Anmeldung über den Menüpunkt Hilfe, HELIUM V Hilfe im HELIUM V Client.

 

Nach dem Einstieg in die Hilfe maximieren Sie bitte das Fenster.

 

Sie finden nun links den Index und die Textsuche .

Durch Klick auf den Index sehen Sie alle Kapitel. Gegebenfalls klappen Sie die Kapitel durch Klick auf auf.

In den Kapiteln können Sie mit Strg+F nach Texten suchen.

 

Sind in den Kapiteln Hinweise auf andere Kapitel enthalten, so sind diese blau hinterlegt. Klicken Sie auf diesen Link und Sie gelangen auf die entsprechende Stelle. Um wieder zurückzugelangen drücken Sie bitte auf den Zurück-Pfeil .

Anwenderspezifische Hilfe

 

Unter dem Menüpunkt Hilfe, Anwenderspezifische Hilfe steht Ihnen ein Link auf Ihre unternehmensspezifische Anwendung von HELIUM V zur Verfügung.

Diese Hilfe ist als HTML Datei aufgebaut. Benutzer die diese Hilfe anzeigen wollen, müssen auf diesen UNC Pfad zumindest Leserechte besitzen.

 

Wird beim Aufruf der anwenderspezifischen Hilfe

angezeigt, so wurde der Pfad nicht definiert.

 

Der Pfad auf Ihre Hilfedatei kann im System unterer Modulreiter Parameter, URL_ANWENDERSPEZIFISCHE_HILFE eingestellt werden.

Bitte achten Sie darauf hier allgemeingültige Pfade einzustellen.

 

Wichtig:

Hier muss eine URL eingegeben werden.

Erscheint

so wurde vermutlich nur ein Filename angegeben. Für den vermutlich üblichen Verweis auf eine Datei muss: file://Server/Freigabe/... angegeben werden. Bitte beachten Sie, dass nur die Vorwärts-Schrägstriche verwendet werden dürfen.

 

Kann die Datei nicht gefunden werden, z.B. wegen Leserechten, so erscheint

.

Bitte entsprechend richtigstellen.
