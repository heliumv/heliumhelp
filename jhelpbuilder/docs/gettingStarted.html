JHelpBuilder - Getting Started

This tutorial will guide you through the process of creating a JavaHelp HelpSet using JHelpBuilder. A basic knowledge of JavaHelp system is required for this tutorial.

It will be helpful to familiarize yourself with the basic UI layout of JHelpBuilder before continuing with this tutorial.

It is recommended that you create a set of HTML files before hand to include in your project. In this tutorial, we will use the files provided in the samples directory.
