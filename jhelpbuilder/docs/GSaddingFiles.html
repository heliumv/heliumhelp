JHelpBuilder - Add Files

Here we will add files to our project.

You can optionally skip this step and add files to the project directory by other means provided by your operating system. However, if you choose to add files to the project outside of JHelpBuilder, please remember to refresh the project directory.

To add files to the project, we go to the FileSystem tab.

  • Go to the FileSystem tab.
  • Select the root node in FileSystem tab.
  • Click the button in the toolbar.
  • In the File Chooser dialog, select the files you want to add to your project. [Files in the samples directory of JHelpBuilder installation.]
  • Click "Add" and the select "Yes".
  • The files are added to the FileSystem tree.
