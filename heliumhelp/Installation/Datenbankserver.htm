Datenbankserver

Von Helium V werden derzeit folgende Datenbankserver unterstützt:

  • PostgresQL Version 8
    Diese Lizenzfreie Version wird mit Ihrer Linux Installation mitgeliefert. In Ihrer MAC Installation wird diese von Apple zur Verfügung gestellt.

  • Mircosoft MSDE Freie Version für Mircrosoft Betriebssysteme.
    Beachten Sie bitte die Größenbegrenzung dieser für den Einsatz mit Microsoft Betriebssystemen kostenlosen Datenbank auf 2GByte.

  • Mircosoft MSSQL Server 2000

  • Microsoft SQL Server 2005 Express

Hinweis:

Von Seiten Helium v werden für den Zugang zum Server nur zwei Benutzer  verwendet, egal wieviele gleichzeitige Benutzer in Ihrem Helium V arbeiten. Beim Einsatz der Datenbankserver ist die zukünftige Größenentwicklung der Datenbank und die eventuellen zusätzlichen auswertenden Zugriffe, welche ev. außerhalb von Helium V von Ihren Mitarbeitern bereitgestellt werden zu beachten. Denken Sie bei der Größenentwicklung auch an die Dokumentenablage.

Auch diese Daten werden derzeit in der Helium V Datenbank abgelegt.
